TyrellSystems PEAR channel

Registering the channel: 
pear channel-discover pear.tyrellsys.com
Listing available packages: 
pear remote-list -c tyrellsys
Installing a package: 
pear install tyrellsys/package_name
Installing a specific version/stability: 
pear install tyrellsys/package_name-1.0.0
pear install tyrellsys/package_name-beta
Receiving updates via a feed: 
http://pear.tyrellsys.com/feed.xml
